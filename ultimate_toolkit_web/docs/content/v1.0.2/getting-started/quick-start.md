# Quick Start

Get Ultimate Toolkit running in 5 minutes.

## Download

Grab the latest installer from the [Releases](https://github.com/Ege-BULUT/ultimate_toolkit_by_egebulut/releases) page and run it.

## First Run

1. Launch Ultimate Toolkit
2. A **floating toolbar** appears at the top of your screen — use it for quick access to floating tools
3. The main window opens to the **Plugin Manager**
4. Toggle the **OCR** or **AI Chat** plugins on
5. Open the **floating window** for any plugin — it stays on top of ALL your windows

## Try It Out

- **OCR**: Open any image, take a screenshot (Win+Shift+S), paste it in OCR
- **AI Chat**: Configure an API key (OpenAI, Anthropic, etc.) and start chatting
- **Floating windows**: Click "Pop Out" in any plugin config to open it as a separate always-on-top window
