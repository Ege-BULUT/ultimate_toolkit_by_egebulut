# Installation

## Pre-built Installer (Recommended)

1. Download the latest `-setup.exe` from [GitHub Releases](https://github.com/Ege-BULUT/ultimate_toolkit_by_egebulut/releases)
2. Run the installer (no admin required)
3. Launch from the Start Menu or desktop shortcut
