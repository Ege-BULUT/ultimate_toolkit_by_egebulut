# Getting Started

Welcome to Ultimate Toolkit! This guide will help you get up and running quickly.

## What is Ultimate Toolkit?

Ultimate Toolkit is an open-source Windows utility toolkit built with Rust and React. It provides a growing ecosystem of plugins for productivity tasks like OCR text extraction, AI chat, and more.

## Prerequisites

- Windows 10 or 11
- For pre-built installers: just download and run
- For building from source: Node.js 18+ and Rust
