# Building from Source

## Prerequisites

- [Rust](https://rustup.rs) (stable toolchain)
- Node.js 18+
- Git

## Steps

```bash
git clone https://github.com/egebulut/ultimate_toolkit_by_egebulut.git
cd ultimate_toolkit_by_egebulut
npm install
npm run tauri:dev
```

For a production build:

```bash
npm run tauri:build
```
