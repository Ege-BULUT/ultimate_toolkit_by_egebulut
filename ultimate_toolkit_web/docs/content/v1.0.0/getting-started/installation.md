# Installation

## Pre-built Installer (Recommended)

1. Download the latest `-setup.exe` from [GitHub Releases](https://github.com/Ege-BULUT/ultimate_toolkit_by_egebulut/releases)
2. Run the installer (no admin required)
3. Launch from the Start Menu or desktop shortcut

> The installer and all releases are auto-built by GitHub Actions — always the latest version.

## What to Expect

When you launch Ultimate Toolkit for the first time:

- The **main window** opens with the Plugin Manager
- A **floating toolbar** (small button bar) appears at the top of your screen — always on top of all windows
- Click toolbar buttons to open **floating windows** (OCR, AI Chat) that stay on top even when the main app is minimized
- The **Welcome screen** introduces key features on your first visit
