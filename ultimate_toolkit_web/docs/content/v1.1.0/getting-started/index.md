# Getting Started

Welcome to Ultimate Toolkit v1.0.1! This guide will help you get up and running quickly.

## What is Ultimate Toolkit?

Ultimate Toolkit is an open-source Windows utility toolkit built with Rust and React. It provides a growing ecosystem of plugins for productivity tasks like OCR text extraction, AI chat, and more.

**What makes it special:**
- **Floating windows** - plugin UIs open as separate always-on-top windows, usable on any screen
- **Modular plugins** - each tool is a plugin you can enable/disable independently
- **Native performance** - built with Tauri (Rust + WebView2), not Electron

## Key Features in v1.0.1

| Feature | Description |
|---------|-------------|
| 🔍 OCR | Extract text from images and screenshots (15+ languages) |
| 🤖 AI Chat | Chat with 7 AI providers from one interface |
| 🪟 Floating windows | Separate OS windows, always on top of all apps |
| 🧰 Floating toolbar | Always-on-top button bar for quick access |
| 🧩 Plugin system | Extend with custom plugins loaded from .js files |

## Prerequisites

- Windows 10 or 11
- For pre-built installers: just download and run
- For building from source: Node.js 18+ and Rust
