# Provider Setup

## Ollama (Local, Free)

1. Download from [ollama.com](https://ollama.com)
2. Run: `ollama pull llama3.2`
3. The app auto-detects Ollama and lists available models

## OpenAI

1. Go to [platform.openai.com/api-keys](https://platform.openai.com/api-keys)
2. Create a new API key
3. Paste it in the AI Chat config

## Anthropic

1. Go to [console.anthropic.com/keys](https://console.anthropic.com/keys)
2. Create an API key (starts with `sk-ant-`)
3. Paste it in the config

## Other Providers

Google Gemini, OpenRouter, DeepSeek, and HuggingFace follow the same pattern: create an API key, paste it in the config.
