# Supported Languages

## Pre-installed

- English (`eng`)

## Available Languages

Additional language packs can be downloaded from the OCR config page:

- Chinese Simplified (`chi_sim`)
- Chinese Traditional (`chi_tra`)
- French (`fra`)
- German (`deu`)
- Japanese (`jpn`)
- Korean (`kor`)
- Spanish (`spa`)
- Turkish (`tur`)
- And more...

## Manual Installation

Language data is stored at `%APPDATA%/ultimate-toolkit/tessdata/`. Download `.traineddata` files from [tesseract-ocr/tessdata_fast](https://github.com/tesseract-ocr/tessdata_fast).
