Features

Everything you need, nothing you don't.

Each tool is a plugin. Toggle it on or off. Configure it your way.

🔍

OCR Text Extraction

Extract text from images, screenshots, and screen regions. Supports 15+ languages with Tesseract OCR. English pre-installed.

Tesseract 15+ Languages Multi-Engine
🤖

AI Chat Assistant

Chat with 7 AI providers from one interface: OpenAI, Anthropic, Gemini, OpenRouter, DeepSeek, HuggingFace, and local Ollama.

7 Providers Ollama Local Floating Window
🧮

Plugin System

Extend the toolkit with community plugins. Each plugin has its own config page and can float independently on your screen. Now with Python plugin support!

Extensible Floating UI Python Scripts
🎨

Beautiful UI

Native Windows feel with premium design. Dark mode, light mode, or system-follow. Smooth animations and hover hints throughout.

Dark Mode System Theme Tooltips
🔄

Auto-Updates

Stays up-to-date automatically. Checks for new versions on startup and can update itself from within the app.

GitHub Releases Self-Update
⚡

Blazing Fast

Built with Rust and Tauri. Native performance with a fraction of the memory usage of Electron-based alternatives.

Native Rust Tauri v2 Low Memory
