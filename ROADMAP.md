# Ultimate Toolkit — Roadmap

## 🎯 Vision
A community-driven Windows utility toolkit where anyone can build and share plugins. Fast, beautiful, and open.

---

## Phase 1: Foundation ⬜ (Current)

> **Goal**: Working desktop app with core plugins

- [x] Tauri + React + Rust project setup
- [x] Plugin system architecture
- [x] Settings (theme, auto-update)
- [x] OCR plugin (Tesseract, 15+ languages)
- [x] AI Chat plugin (7 providers, Ollama auto-detect)
- [x] Floating window UI for plugins
- [ ] GitHub repository + CI/CD
- [ ] First release (v0.1.0)
- [ ] Landing page on Vercel

## Phase 2: More Plugins 🔵

> **Goal**: Expand utility library

| Plugin | Description | Priority |
|--------|-------------|----------|
| Color Picker | Screen eye dropper + format conversion | High |
| Screen Ruler | Pixel measurement tool | High |
| Find My Mouse | Ctrl-highlight cursor locator | Medium |
| Batch Rename | Mass file renamer with patterns | Medium |
| Keyboard Manager | Custom global shortcuts | Medium |
| Clipboard History | Searchable clipboard manager | Low |
| Window Manager | Snap layouts + window organizer | Low |

## Phase 3: Ecosystem 🟢

> **Goal**: Community + cross-platform

- Plugin SDK + CLI (`utoolkit create-plugin`)
- Community plugin repository
- One-click plugin install from marketplace
- Localization (i18n): Turkish, German, French, Spanish, Japanese, Korean, Chinese
- macOS support (Tauri makes this feasible)
- Linux support

## Phase 4: Advanced 🟣

> **Goal**: Power features

- Workflow automation (trigger chains)
- Script runner (Python, JS snippets)
- System monitor (CPU, RAM, disk)
- AI-powered smart actions
- Theme marketplace
